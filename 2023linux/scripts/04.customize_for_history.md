### customize for history

```
 echo 'export HISTTIMEFORMAT="%d/%m/%y/%T "' >> ~/.bash_profile
```


  #### e.g - example 
  
  ``` 
  143  18/03/23/08:16:15  vi .bash_profile
  144  18/03/23/08:16:40  source .bash_profile
  145  18/03/23/08:16:42  history
  146  18/03/23/08:16:52  vi .bash_profile
  147  18/03/23/08:17:16  source .bash_profile
  148  18/03/23/08:17:18  history
  149  18/03/23/08:17:33  vi .bash_profile
  150  18/03/23/08:17:42  source .bash_profile
  151  18/03/23/08:17:43  history
  ```
